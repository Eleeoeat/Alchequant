# Alchequant 启动说明

## 1. 环境要求

- Python 3.10 到 3.12
- Windows、macOS、Linux 均可运行
- 推荐先创建虚拟环境，再安装依赖

## 2. 安装依赖

```bash
python -m pip install -r requirements.txt
```

如果你的系统默认命令是 `python3`：

```bash
python3 -m pip install -r requirements.txt
```

## 3. Windows 启动

双击 `run.bat`。

也可以在命令行运行：

```bash
python -m streamlit run app.py --server.port 8501
```

## 4. macOS / Linux 启动

```bash
bash run_mac.sh
```

或直接运行：

```bash
python3 -m streamlit run app.py --server.port 8501
```

## 5. 打开平台

启动后访问：

```text
http://localhost:8501
```

## 6. 常见问题

### 找不到 Python

请先安装 Python 3.10+，或激活你的虚拟环境。

### 找不到 Streamlit

说明当前 Python 环境没有安装依赖，运行：

```bash
python -m pip install -r requirements.txt
```

### 没有数据

发布版默认包含 `data/stocks.db` 示例数据库。若该文件缺失，平台仍能启动，但股票列表和分析结果会为空。

可通过以下命令重新下载数据：

```bash
python scripts/download_data.py
```

该命令需要联网。

### AI 增强报告无法使用

AI 增强是可选功能。没有 API Key 时，系统会使用本地规则化报告文字，不影响回测、图表和 HTML 报告生成。
